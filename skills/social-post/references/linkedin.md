# LinkedIn Platform Reference

## Format Constraints

- **Character limit**: 3,000 chars total; sweet spot is 900–1,500
- **Visible before "See more"**: ~140 chars on desktop, ~110 on mobile — this is the hook
- **Hashtags**: 3 max. More than 3 triggers engagement penalties.
- **Links**: Do NOT put the link in the post body. LinkedIn's algorithm deprioritizes posts with outbound links. Put the link in the first comment immediately after publishing.
- **Images**: Optional but helpful; 1200×675px

## Content Best Practices

- **Open with a hook**: The first 140 chars must make someone stop scrolling. Specific > vague. Problem statement or counterintuitive observation work well.
- **Tone**: Thought leadership, not marketing. Write as "here's what I learned," not "check out my tool."
- **Structure that works**: Hook → problem or moment that prompted this → what you did or found → key insight → soft CTA
- **Comments > likes**: The algorithm weights comments 15x more than likes. Ask a genuine question at the end to invite response.
- **Early engagement matters**: The first 90 minutes after posting determine reach. Don't post and disappear.

## Anti-Patterns

- Link in the post body (kills reach — always put it in the first comment)
- Generic advice ("Here are 5 things I learned about AI...")
- Hype punctuation ("This CHANGED Everything!!!")
- Engagement bait ("Comment YES if you agree")
- Vague opening that doesn't hook immediately
- Generic CTAs ("Feel free to check it out and let me know your thoughts!" / "Links in the comments!")
- The standard launch post formula: problem setup → tool description → key feature → CTA. Reads as a template, not a person. DvQ prefers casual and personal — fewer words, more direct, sounds like something you'd actually say to a colleague.

## Structure Template

```
[Hook — specific problem or observation, ≤140 chars]

[1–2 sentences: personal context — when did this come up, what made it real]

[What you built/found/decided, and why]

[Key insight or takeaway — the thing worth reading the whole post for]

[Soft CTA — invites engagement without demanding it]
```

## Hashtag Selection

Choose 3 that are specific enough to reach the right audience but broad enough to have traffic:
- For dev tools: #claudecode #aitools #developertools
- For open-source: #opensource #buildinpublic
- For productivity: #productivity #aiworkflow

## Output Checklist

- [ ] Hook is ≤140 chars and specific
- [ ] No link in post body
- [ ] 3 hashtags (no more)
- [ ] First comment drafted with the link
- [ ] Tone is personal/candid, not promotional
- [ ] 900–1,500 chars total
